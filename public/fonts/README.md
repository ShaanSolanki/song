# Satoshi fonts

RESONANCE uses **Satoshi** (Black / Bold / Medium / Regular). Drop the `.woff2`
files here so headings render in the intended typeface:

```
public/fonts/Satoshi-Black.woff2
public/fonts/Satoshi-Bold.woff2
public/fonts/Satoshi-Medium.woff2
public/fonts/Satoshi-Regular.woff2
```

Get them (free) from Fontshare: https://www.fontshare.com/fonts/satoshi

If these files are absent the site falls back to the system sans-serif stack —
everything still works, it just won't be pixel-identical to the brief.
